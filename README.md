# Node.js Express Authentication API

A robust, production-ready authentication system built with **Node.js**, **Express**, and **MongoDB**. This project implements modern security best practices including JWT (JSON Web Tokens), password hashing, and global error handling.

---

## üöÄ Features
- **User Authentication**: Secure Signup, Login, and Logout logic.
- **Security**: Password hashing with `bcryptjs`.
- **Authorization**: Protected routes using JWT tokens in headers.
- **Error Handling**: Global middleware to catch and format API errors.
- **Environment Management**: Secure configuration using `.env`.
- **Scalable Architecture**: Modular folder structure (Controllers, Routes, Models, Middlewares).
